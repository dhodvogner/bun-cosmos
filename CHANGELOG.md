# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-19

### Added
- Initial release of bun-cosmos as a proper Bun library
- Main API: `startBunCosmos()` function for quick setup
- Class API: `BunCosmos` class for advanced control
- Configuration system with TypeScript types
- Support for Bun build plugins
- Hot Module Replacement (HMR) with automatic browser reload
- Comprehensive documentation and README
- Example project demonstrating usage
- Automated tests with Bun test runner
- GitHub Actions for CI/CD
- MIT License

### Features
- Zero-config setup with sensible defaults
- Configurable ports for all servers
- Custom fixtures directory support
- Source map support for debugging
- Watch mode with automatic rebuilds
- Unified proxy server interface
- TypeScript native implementation
